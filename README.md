## Philosophies
- ***Lighting fast***
- ***Less dependencies***
- ***No `unsafe {}`, `.unwrap()`***
- ***Easy to use***
- ***Strive for excellence***

## Documentations
```sh
# Use `--help` flag to get more information.
subalfred --help
# Each subcommand also provides its own `--help`.
subalfred check --help
subalfred check runtime --help
```
